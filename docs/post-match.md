# Post-Match

## End Conditions
Match ends when the 3 minute timer expires or the technician stops it early.

---

## Technician

1. Stop all robot units without disturbing game element positions
2. Anything after the timer expires or after the technician begins stopping the robot does not count
3. Retrieve robot after referee signals
4. Slow or complicated retrieval results in a **Yellow Card**

---

## Auto-Shutoff
Build a mechanism to stop the robot automatically at 3 minutes. Manual stopping is allowed but if the robot is mid-interaction when stopped it can cause scoring complications. Auto-shutoff eliminates this risk.

---

## Referee Steps
1. Finalize score
2. Signal technician to retrieve robot
3. Field personnel reset for next match

---

## Emergency Stop
Robot must be stoppable via emergency stop button at any time.
