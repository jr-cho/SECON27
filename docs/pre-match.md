# Pre-Match

## Field Personnel Checklist

- [ ] Field set up per Figure 3.2.1
- [ ] Die rolled, pit spot determined (see pit-stop.md for die roll table)
- [ ] Correct Apriltag (1, 2, or 3) fastened to wall near start area
- [ ] Two obstacle cars placed on course, one in inner lane and one in outer lane, each on a long straightaway
- [ ] Green start flag secured in the **down** position
- [ ] All field elements within ± 1 in. position tolerance

---

## Technician Checklist

- [ ] Robot placed completely inside the 12x12x12 in. start box
- [ ] Tire pre-loaded onto robot
- [ ] All settings confirmed
- [ ] Robot declared ready to referee

---

## Rules

Do not touch the field or game elements after randomization. Once obstacles and the Apriltag are placed, any interaction voids scoring for all tasks tied to that randomization (obstacles and Apriltag spot).

Technician may only interact with the robot within the 12 in. start box during pre-match. Touching any other part of the field or game elements is a Red Card.

---

## Apriltag
- Mounted on wall near the start area
- Displays pit spot ID: 1, 2, or 3
- Tag family: 41h12 | Size: 68mm
