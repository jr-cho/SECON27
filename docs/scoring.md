## Notes

### Speed Bonus
1 point per full 10 second interval under 3:00.

| Finish Time | Bonus Points |
|---|---|
| 2:54 | 0 |
| 2:49 | 1 |
| 2:39 | 2 |
| 2:29 | 3 |

### Pit Stop
- Full 15 points if robot stops for 2 or more seconds
- Only 10 points if robot leaves before 2 seconds
- Must happen between lap 1 and 2, or lap 2 and 3

### Tire Exchange
- Robot carries a tire from race start
- Drop it in the pit area during the pit stop
- Pick up a different tire, cannot reuse the one dropped
- Both actions score independently

### Clean Lap Bonuses
- Awarded per lap, not per race
- Wall and obstacle bonuses are separate
- Maximum: 3 laps x (10 + 15) = 75 points

### Flag Start Bonus
- 10 points for starting on the green flag
- 0 points if robot starts early, no additional deduction

### "In" Rule
More than **75%** of the robot must be inside the pit spot boundary. No partial credit.
