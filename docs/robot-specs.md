# Robot Specs

## Starting Size
Must fit completely inside a **12 in. x 12 in. x 12 in.** cube at match start. Robot can expand beyond this once the match begins.

---

## Autonomy
Fully autonomous during the match. No external commands, signals, or remote inputs of any kind once the match starts.

Allowed inputs:
- Pre-programmed instructions
- Onboard sensor data (camera, IR, encoders, etc.)

---

## Start Trigger
Robot must detect the green start flag being raised. A manual start button is allowed as a backup only.

---

## Auto-Shutoff
Build a mechanism to stop the robot automatically at 3 minutes. Manual stopping is allowed but causes scoring complications if the robot is mid-action when time expires.

---

## Emergency Stop
Robot must have an emergency stop button for safe post-match retrieval.

---

## Tire Carry
Robot must carry a tire from race start through the pit stop. The technician places it on or in the robot before the match starts.

---

## Flag Mechanism
Robot must be able to raise a school flag, logo, or emblem during the final lap for the 25 point bonus.

---

## Inspection
All team game elements must pass inspection before use. Subject to GM1 section 5.2.1 mechanical parts and materials rules.

---

## Field Boundary
Robot must stay on the field at all times. Reaching outside the perimeter results in a Yellow Card. Intentional violations can result in a Red Card.
