# Match Play

## Start
Match begins when the referee raises the green start flag and declares the match started.

- Robot should detect the flag autonomously
- If the robot does not respond, the technician may press the manual start button
- Starting before the flag eliminates the 10 point flag start bonus

---

## Race Flow

```
Start Box
   |
Enter course (+25 pts)
   |
Lap 1
   |
[Pit stop here, or after lap 2]
   |
Lap 2
   |
[Pit stop here, if not done after lap 1]
   |
Lap 3 — raise school flag for +25 pts
   |
Cross finish line
```

---

## Lap Rules
- 3 laps total required
- Must complete all 3 laps within **3 minutes** to earn the 100 point completion bonus
- Each lap scored independently for clean lap bonuses:
  - No wall contact: +10 pts
  - No obstacle contact: +15 pts

---

## Flag Raising
- School flag, logo, or emblem must be raised during the final lap
- Worth 25 points
- Must be a physical flag raised by the robot

---

## Timer
- 3 minutes from match start
- No extensions for late starts
- Actions after time expires do not count
- Field damage after time still results in a Yellow Card

---

## Autonomous Operation
Robot must operate fully on its own once the match starts. No external signals or commands permitted.
